# 论文资料下载说明

本目录包含根据项目源码、数据库初始化脚本和前端菜单配置整理的论文资料。

> 说明：当前仓库不支持提交二进制文件，因此不再保存 `.docx` 文件。请使用下方的 Word 兼容 HTML 文件，在 Microsoft Word / WPS 中打开后另存为 `.docx`。

## 文件列表

- `thesis_data_summary.md`：Markdown 源文件，便于在仓库页面直接预览和修改。
- `thesis_data_summary_word.html`：Word 兼容 HTML 文件，可用 Word / WPS 打开并另存为 `.docx`。

## 生成 Word 文档的方法

1. 在仓库页面下载或打开：

   ```text
   docs/thesis_data_summary_word.html
   ```

2. 使用 Microsoft Word 或 WPS 打开该 HTML 文件。
3. 选择“文件” → “另存为”。
4. 文件类型选择 `.docx`，即可得到 Word 文档。

如果只需要在线预览内容，可直接打开：

```text
docs/thesis_data_summary.md
```
