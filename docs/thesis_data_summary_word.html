基于 Spring Boot 与 Vue 的便利店管理系统论文数据整理

本文档根据项目源码、数据库初始化脚本和前端菜单配置整理，可用于论文的系统概述、开发环境、需求分析、数据库设计、测试数据和实验数据章节。

系统名称建议：基于 Spring Boot 与 Vue 的便利店管理系统设计与实现。

系统定位：本项目是面向便利店日常经营的管理系统，覆盖商品、供应商、采购、收银、员工、消费者和基础字典等业务。

一、开发环境与技术栈

类别技术/版本论文表述建议
后端框架Spring Boot 2.2.2.RELEASE采用 Spring Boot 快速搭建后端服务。
JDKJava 1.8系统运行环境为 JDK 1.8。
持久层MyBatis + MyBatis-Plus用于数据访问层开发和通用 CRUD。
数据库MySQL 5.7 / bianlidian使用 MySQL 存储系统业务数据。
权限认证Apache Shiro 1.3.2用于登录认证、角色权限控制。
前端框架Vue 2.6.10前端采用 Vue 构建后台单页管理界面。
UI 框架Element UI 2.13.0使用 Element UI 完成后台页面组件化设计。
图表工具ECharts 4.6.0用于收银报表和统计图表展示。

三、系统角色与功能模块

角色功能模块主要操作
管理员基础数据管理查看、新增、修改、删除商品类型、供应商类型、地区类型、信用等级等字典数据。
管理员供应商信息管理查看、新增、修改、删除供应商基础资料。
管理员商品管理查看、新增、修改、删除商品，维护库存、进价、售价和上下架状态。
管理员采购管理查看、新增、删除采购单及采购明细。
管理员收银管理查看、新增、删除收银订单，查看收银明细并生成报表。
管理员员工管理查看、新增、修改、删除员工账号资料。
管理员消费者管理查看、新增、修改、删除消费者账号资料。
员工供应商/商品查看查看供应商信息和商品信息。
员工收银管理新增收银订单，查看和删除收银记录。
消费者商品浏览购买查看商品、购买商品、支付订单。
消费者消费记录查询查看历史消费记录。

十一、可直接写入论文的数据规模描述

本系统以便利店日常经营业务为研究对象，数据库初始化样本包含 3 条商品数据、3 条供应商数据、3 条员工数据、3 条消费者数据、1 条管理员数据、20 条字典数据、15 条收银订单数据、24 条收银明细数据以及 5 条采购明细数据。商品初始化库存总量为 965 件，库存按进价估算总成本为 154725.00 元，按售价估算总价值为 209105.00 元。收银模块初始化订单总数为 15 单，销售总额为 4620.00 元，其中线上支付 9 单、线下支付 6 单。系统数据能够覆盖商品维护、供应商管理、采购入库、收银结算、消费记录查询等核心业务场景。
